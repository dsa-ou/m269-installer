# Using the software (shortcut)

These instructions are for **Windows** only.

1. Double click the `M269-start` desktop shortcut created by the installation script.
   A PowerShell window opens where Jupyter writes its log messages.
2. Once only: on **Windows 11**, if you're asked to 'select an app to open this HTML file', choose your usual browser and click 'always'.
3. After your browser opens JupyterLab, listing the contents of your M269 folder,
   double-click subfolder `notebooks`, then file `M269.ipynb`.
   This opens the M269 book with its table of contents.

> **Info:** You may also open M269 notebooks [from a PowerShell](use-terminal.md)
> but it's less convenient.
